## Project Description

The Interactive Quiz Application is an educational tool designed to enhance user knowledge through engaging, timed quizzes on various topics. Developed during a four-week internship at CODTECH IT SOLUTIONS Private Limited, it emphasizes front-end development skills.

The app presents dynamic questions from a pool, with multiple-choice options and a 30-second timer per question. Scoring rewards accuracy, and feedback helps learning. User flow includes difficulty selection, quiz progression with progress bars, results display, and leaderboard storage.

Technologies: HTML5 for structure, CSS3 for glassmorphism styling, gradients, animations, and responsive design using Flexbox/Grid; JavaScript (ES6+) for logic, timers, scoring, and localStorage. No external libraries beyond Font Awesome CDN.

UI/UX focuses on engagement with translucent cards, smooth transitions, dark mode, sound effects, and mobile responsiveness.

Key features: countdown timer, instant feedback, persistent leaderboard, accessibility.

Challenges: Timer inconsistencies due to interval stacking, resolved with proper setInterval/clearInterval; UI scaling issues fixed with responsive layouts and larger containers.

Learnings: Asynchronous programming, responsive design, iterative refinement in front-end development. (Word count: 178)

## Features

- **Dynamic Questions:** Randomized questions from a pool with varying difficulty levels.
- **Timer Functionality:** 30-second countdown per question with visual progress bar.
- **Scoring System:** Points awarded for correct answers, with bonuses for speed.
- **Instant Feedback:** Visual and auditory cues for correct/incorrect responses.
- **Leaderboard:** Persistent storage of high scores using localStorage.
- **Dark Mode Toggle:** Switch between light and dark themes for user preference.
- **Sound Effects:** Audio feedback for interactions and time warnings.
- **Responsive Design:** Adapts to different screen sizes and devices.
- **Accessibility:** Keyboard navigation and semantic HTML structure.
- **Animations:** Smooth transitions, confetti for high scores, and hover effects.

## Tech Stack

- **HTML5:** For semantic structure and content organization.
- **CSS3:** For styling, animations, and responsive layouts.
- **JavaScript (ES6+):** For logic, interactivity, and data handling.
- **Font Awesome:** For icons (via CDN).

## How to Run the Project

1. Ensure you have a modern web browser (e.g., Chrome, Firefox, Edge) and Python installed.
2. Download or clone the project files to your local machine.
3. Open a terminal (e.g., PowerShell or Command Prompt) and navigate to the project directory:
   ```
   cd "P:\CODETECH IT SOLUTIONS\TASK1"
   ```
4. Start a local HTTP server:
   ```
   python -m http.server 8000
   ```
5. Open your web browser and go to `http://localhost:8000`.
6. The application will load, and you can start the quiz by selecting a difficulty level.
7. Alternatively, directly open `index.html` in your browser (no server needed for basic functionality).
